1. 计算公式
局部莫兰指数 $I_i$ 的计算公式为：

$
I_i = \frac{(x_i - \bar{x})}{S^2} \cdot \left( \sum_{j=1,j \ne i}^{n} w_{ij}(x_j - \bar{x}) \right)
$

其中：

$x_i$：目标单元 $i$ 的属性值。
$\bar{x}$：所有单元属性的均值。
$S^2$：属性值的方差,这玩意还不是方差？（$S^2 = \frac{\sum_{j=1,j \ne i}^{n}(x_j - \bar{x})^2}{n-1}$）。
$w_{ij}$：空间权重矩阵（如邻接矩阵、距离衰减权重等）。
$n$：总单元数。
